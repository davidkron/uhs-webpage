UHS - Unified Header Source

UHS (Unified Header Source), is an abstractions layer on top of C++ more complicated header and source files. UHS provides C++ developers with one single coding interface; the UHS file.

UHS is a simple visual studio package that automatically generates CPP and H files for you while you're coding. Things that should go to the H files (declarations, macros, templates, enums) is put into the H files while things that should be in the CPP files (implementation and definitions) is put in the CPP files.
